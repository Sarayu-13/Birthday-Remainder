# 🎀 Birthday Reminder

A cute and girly birthday reminder app built with **Vite + React**. Never forget a friend's special day again! 💕

## ✨ Features

- 🎂 Add birthdays with name, date, and a cute emoji avatar
- 📅 Birthdays sorted by who's coming up next
- 🎉 Special glowing card + confetti banner for today's birthdays
- ⏰ Countdown showing days remaining
- ♈ Auto-detected zodiac sign for each birthday
- 🔍 Search friends by name
- 💾 Data saved in localStorage — persists across sessions
- 💖 Girly pink/purple aesthetic with smooth animations

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or above)

### Installation

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/birthday-reminder.git

# Navigate into the project
cd birthday-reminder

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open your browser at `http://localhost:5173`

## 🏗️ Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder.
